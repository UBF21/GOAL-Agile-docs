# Nivel 3 — GOAL Program Lead (GPL)

## Descripción General

El GOAL Program Lead es la certificación de nivel practicante más alto en el sistema GOAL. Valida la competencia en operar el framework GOAL a **nivel de programa** — específicamente, coordinar múltiples equipos operando GOAL simultáneamente, gestionar dependencias entre equipos y liderar eventos y estructuras de gobernanza a nivel de programa.

---

## Prerequisitos

**Certificación previa:** Certificación GFMC activa (no expirada al momento de la solicitud GPL)

**Experiencia multi-equipo:** Un mínimo de doce (12) meses de experiencia trabajando en múltiples equipos en un rol de coordinación. Esta experiencia debe implicar gestión real de dependencias entre equipos y planificación a nivel de programa.

**Ciclos de programa:** Un mínimo de cuatro (4) ciclos de programa completos coordinados a nivel de programa. Un "ciclo de programa" se define como un ciclo completo donde el candidato fue responsable de facilitar el Program Sync, mantener el Dependency Register y co-facilitar el Program Retrospective para un grupo de al menos dos equipos GOAL.

**Portfolio de Programa:** Los candidatos deben presentar y tener aceptado un Portfolio de Programa antes de presentarse al examen.

---

## Dominios de Conocimiento

| Dominio | Descripción | Peso en el Examen |
|---------|-------------|:-----------------:|
| Dominio 1: Arquitectura de GOAL de Programa | Niveles de escalado, artefactos a nivel de programa, estructuras multi-equipo | 15% |
| Dominio 2: Roles a Nivel de Programa | Autoridad del Program Flow Master y Program Strategist, límites y ejecución | 15% |
| Dominio 3: Gestión de Dependencias | Identificación, seguimiento, escalación y resolución estructural de dependencias entre equipos | 25% |
| Dominio 4: Métricas a Nivel de Programa | Cross-Team Block Rate, Tasa de Dependencias, Program Goal Completion Rate, interpretación | 20% |
| Dominio 5: Eventos de Coordinación Multi-Equipo | Program Sync, Cross-Team Dependency Review, Program Retrospective | 15% |
| Dominio 6: Anti-Patrones de Programa y Límites de Escala | Reconocer anti-patrones de escalado, saber cuándo el modelo de escalado de GOAL alcanza sus límites | 10% |

## Portfolio de Programa

### Sección A — Registros de Ciclo de Programa (requerido)
Resúmenes de un mínimo de 3 de los 4 ciclos de programa requeridos, cada uno incluyendo:
- Equipos involucrados (número y tamaño aproximado)
- Objetivos de Programa para el ciclo
- Resumen del Dependency Register
- Cross-Team Block Rate y tiempo promedio de resolución
- Un desafío de coordinación entre equipos significativo y cómo se resolvió
- Una mejora de nivel de programa de la Retrospectiva y su resultado

Cada resumen: 400–600 palabras.

### Sección B — Evidencia de Escalación de Dependencias (requerido)
Documentación de al menos 2 situaciones reales en las que una dependencia entre equipos escaló a En Riesgo o Bloqueada:
- La descripción de la dependencia
- La ruta de escalación seguida
- El resultado de la resolución
- Qué cambio estructural (si hubo alguno) se hizo para prevenir la recurrencia

Cada relato: 400–600 palabras.

### Sección C — Análisis de Métricas de Programa (requerido)
Un análisis escrito de las métricas entre equipos de un programa real durante al menos 3 ciclos de programa consecutivos. Longitud: 800–1.200 palabras.

### Sección D — Evidencia de Program Retrospective (requerido)
Resumen de un Program Retrospective facilitado por el candidato. Longitud: 400–500 palabras.

## Formato del Examen

**Examen escrito:**

| Componente | Detalle |
|------------|---------|
| Tipo de examen | Basado en ordenador, supervisado, libro cerrado |
| Número de preguntas | 130 |
| Duración | 180 minutos |
| Opción múltiple (1 correcta) | 80 preguntas |
| Selección múltiple (2+ correctas) | 35 preguntas |
| Basadas en escenarios con análisis | 15 preguntas |

**Panel Review (después de aprobar el examen escrito):** Evaluación en vivo de 60 minutos realizada por un panel de dos GPLs certificados.

---

## Criterios de Aprobación

| Componente | Mínimo Requerido |
|------------|:----------------:|
| Puntuación general del examen | 80% (104 de 130 preguntas) |
| Todos los dominios | 70% mínimo por dominio |
| Panel Review | Aprobado (evaluación de rúbrica) |
| Portfolio de Programa | Aceptado (previo al examen) |

---

## Validez y Renovación

**Validez:** 2 años

**Renovación:** 40 CEUs dentro del período de 2 años, con al menos 20 CEUs de contenido designado GPL. Presentar un Resumen de Actividades confirmando práctica activa continua a nivel de programa.

**Expirada:** Debe repetir el examen completo de GPL y presentar un Portfolio de Programa actualizado.
