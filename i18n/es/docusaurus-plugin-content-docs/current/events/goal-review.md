# Goal Review

La Goal Review es el evento GOAL celebrado al final de cada Ciclo de Objetivos, antes de la Retrospectiva Basada en Datos. Evalúa si el ciclo alcanzó sus objetivos y mide el valor entregado.

---

## Descripción General del Evento

| Elemento | Detalle |
|---------|---------|
| Duración | 30–45 minutos |
| Facilitador | Product Strategist |
| Participantes | Todos los roles + stakeholders relevantes |
| Frecuencia | Al final de cada Ciclo de Objetivos |

---

## Agenda (4 Pasos)

### Paso 1 — Revisión de objetivos primarios (20 min)

Para cada objetivo primario definido en el Goal Cycle Plan:
- ¿Se completó? ¿Cumple la DoD?
- ¿Cuál es la evidencia del valor entregado?
- Si no se completó, ¿cuál es el estado y el plan?

### Paso 2 — Revisión de objetivos secundarios (5 min)

Para cada objetivo secundario:
- ¿Completado o diferido?
- Si se diferió, ¿va al próximo ciclo?

### Paso 3 — Revisión del Cycle Accuracy Index (5 min)

El Flow Master registra la duración real vs estimada. Este dato se añade al Cycle Accuracy Index para el seguimiento a largo plazo.

### Paso 4 — Resumen del valor entregado (10 min)

El Product Strategist presenta la evidencia del valor entregado usando el Framework de Valor de 3 Capas:
- **Capa 1 (Valor de Negocio):** Impacto en KPIs — revenue, churn, coste, cumplimiento
- **Capa 2 (Valor de Usuario):** Mejoras en la experiencia del usuario
- **Capa 3 (Valor Técnico):** Mejoras en la capacidad de entrega futura

---

## Lo que la Goal Review NO es

- Una evaluación o puntuación del equipo
- Una reunión de demostración que sustituye a la entrega real
- Opcional — se celebra después de cada ciclo sin excepción
- El momento para debatir si los objetivos eran correctos (eso va en la retrospectiva)

---

## Comparación con el Sprint Review de Scrum

| Dimensión | Sprint Review de Scrum | Goal Review de GOAL |
|-----------|----------------------|-------------------|
| Foco | Mostrar trabajo completado | Evaluar si se alcanzaron los objetivos |
| Input del stakeholder | Feedback del producto | Evidencia de valor verificada |
| Datos revisados | Burndown del sprint | Cycle Accuracy Index, evidencia del valor |
| Responsabilidad | Items del sprint completados | Objetivos del ciclo alcanzados |
