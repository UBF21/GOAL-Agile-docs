# Daily Flow Sync

El Daily Flow Sync es el reemplazo de GOAL para el Daily Standup de Scrum. Es una reunión diaria de 10–12 minutos enfocada en la salud del flujo de trabajo — no en informes individuales de estado.

---

## Descripción General del Evento

| Elemento | Detalle |
|---------|---------|
| Duración | 10–12 minutos exactos |
| Facilitador | Flow Master |
| Participantes obligatorios | Flow Master, Delivery Team |
| Frecuencia | Cada día laboral del ciclo, sin excepción |
| Puede reemplazarse por | Nada — ni mensaje asíncrono, ni chat |

---

## Las Tres Preguntas

El Daily Flow Sync aborda tres preguntas, revisando el tablero de **derecha a izquierda** (desde Done hasta Ready):

**Pregunta 1:** ¿Qué avanzó desde ayer?
- Se trata de transiciones de columna, no de actividad
- "Terminé código X" no es una respuesta de flujo — "X se movió de In Progress a Code Review" sí lo es

**Pregunta 2:** ¿Qué está bloqueado o en riesgo de bloquearse?
- Si una tarea está bloqueada y no está en la columna Blocked, se mueve ahora mismo
- Si una tarea está "en riesgo" pero no bloqueada formalmente, se identifica para monitoreo

**Pregunta 3:** ¿Qué está ralentizando el flujo?
- ¿Hay acumulación en Code Review?
- ¿Está el WIP cerca del límite?
- ¿Hay tareas con indicador de envejecimiento?

---

## Lo que el Daily Flow Sync NO es

- Una reunión de reporte de estado individual
- Un lugar para resolver problemas técnicos (eso ocurre después)
- Un lugar para planificación de tareas
- Reemplazable por un mensaje de Slack o actualización asíncrona

---

## Cuando el Sync Excede 12 Minutos

Si una discusión excede los 12 minutos, el Flow Master la interrumpe con:

> "Esto necesita más tiempo del que tenemos ahora. [Nombres específicos], quedémonos 5 minutos después para resolverlo."

El sync no se extiende. La discusión continúa en una conversación separada.

---

## Comparación con el Scrum Standup

| Dimensión | Scrum Daily Standup | GOAL Daily Flow Sync |
|-----------|--------------------|--------------------|
| Foco | Reporte individual de actividad | Salud del flujo del sistema |
| Preguntas | "¿Qué hiciste? ¿Qué harás? ¿Impedimentos?" | "¿Qué se movió? ¿Qué está bloqueado? ¿Qué ralentiza el flujo?" |
| Dirección del tablero | No especificada | Derecha a izquierda |
| Cuando se salta | Común en la práctica | Violación del proceso |
| Resolución de problemas | A menudo ocurre en la reunión | Explícitamente fuera de la reunión |
